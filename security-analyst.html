Security Analyst

SecureNet Corp. • Washington, DC

Full-time

Job Description

Join our cybersecurity team to protect our organization's digital assets. You'll monitor systems, investigate incidents, and implement security measures to defend against threats.

Responsibilities:

  • Monitor security alerts and investigate potential incidents
  • Perform vulnerability assessments and penetration testing
  • Develop and enforce security policies and procedures
  • Conduct security awareness training for employees
  • Stay updated on emerging threats and security technologies

Requirements:

  • 3+ years of experience in information security
  • Knowledge of network security, firewalls, IDS/IPS
  • Familiarity with security frameworks (NIST, ISO 27001)
  • Experience with security tools (SIEM, vulnerability scanners)
  • Relevant certifications (Security+, CISSP, CEH) preferred
