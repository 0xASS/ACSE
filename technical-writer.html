Job Description

We're looking for a Technical Writer to create clear, comprehensive documentation for our developer‑facing APIs and software products. You'll work closely with engineering and product teams.

Responsibilities:

  • Write API documentation, user guides, and tutorials
  • Collaborate with developers to understand technical details
  • Maintain and update existing documentation
  • Create diagrams, screenshots, and other visual aids
  • Ensure consistency and clarity across all documentation

Requirements:

  • 3+ years of technical writing experience in software
  • Ability to explain complex technical concepts simply
  • Experience with documentation tools (Markdown, Git, ReadTheDocs)
  • Familiarity with software development processes
  • Excellent written and verbal communication skills
